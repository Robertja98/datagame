# DataGame

A Phaser 3 + TypeScript + Vite browser game prototype that mixes arcade gameplay inspired by Frogger, Galaga, and Centipede with Grade 12 Data Management questions.

## Current Prototype

- Cross enemy lanes to reach the correct answer gate.
- Shoot obstacles to open a path.
- Answer data management questions to score points.
- Built with code-generated shapes, so no art assets are required yet.

## Stack

- Phaser 3
- TypeScript
- Vite

## Getting Started

Node.js is required to run this project locally.

1. Install Node.js 20 or newer.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local Vite URL in your browser.
